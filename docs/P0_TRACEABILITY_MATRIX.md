## Notes

- The current UI uses `/measure/*` as canonical runtime routes and keeps `/metering/*` as compatibility aliases.
- PDF, login/RBAC, multi-tenancy, async queues, and real payment/bank/tax/e-sign integrations are P1 or out of scope.
- All user-facing evidence must retain the simulation-reference, non-legal-settlement boundary.
