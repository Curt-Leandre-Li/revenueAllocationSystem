# 系统测试方案

## 当前状态

本文件为 V1.2 测试方案归档摘要。当前轮次只做文档更新，不写测试代码。

## 测试范围

- 数据接入管理。
- 数据资源管理。
- 参与方管理。
- 质量评估管理。
- 数元计量管理。
- 贡献度与效用计算。
- MD-DShap 计算管理。
- 收益分配模拟。
- 合同约束管理。
- 报告生成与导出。
- 参数配置。
- 用户与权限管理（P1）。
- 审计日志管理。

## 关键验收

- MD-DShap 默认启用。
- 基础 Shapley 仅作 `baseline_check`。
- P0 不要求登录/RBAC、PDF、异步任务。
- P0 导出 Markdown/CSV/JSON/JSONL。
- 导出文件包含字段清单、`report_id`、`checksum` 和免责声明。
- 计算、重新计算、导出、锁定均可审计。
- 结果仅为模拟参考，非法律结算 / 非法定结算结果。

详细验收见 `docs/acceptance_checklist.md`。
