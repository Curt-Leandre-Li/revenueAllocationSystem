# 系统测试方案

## 当前状态

本文件为 V1.2 测试方案归档摘要。当前轮次同步合同比例主流程、P0/P1 边界和必要回归验证。

## 测试范围

- 数据接入管理。
- 数据资源管理。
- 参与方管理。
- 质量评估管理。
- 数元计量管理。
- 贡献度与效用计算。
- MD-DShap 计算管理。
- 合同分配规则。
- 收益分配模拟。
- 报告生成与导出。
- 参数配置。
- 用户与权限管理（P1）。
- 审计日志管理。

## 验收重点

- MD-DShap 默认启用。
- 基础 Shapley 仅作 `baseline_check`。
- P0 不要求登录/RBAC、PDF、异步任务。
- P1 本地登录、RBAC、PDF、模板导入、任务状态和历史报告只能作为本地演示扩展验收，不得写成生产认证或 P0 必须能力。
- P0 导出 Markdown/CSV/JSON/JSONL。
- 导出文件包含字段清单、`report_id`、`checksum` 和免责声明。
- 计算、重新计算、导出、锁定均可审计。
- 结果仅为模拟参考，非法律结算 / 非法定结算结果。

详细验收见 `docs/acceptance_checklist.md`。
