# 研发产出物清单

## Document Purpose

This directory records formal R&D archive deliverables for the data revenue allocation system before coding starts. These files support software-copyright, internal review, and later implementation handoff.

The files are archive deliverables, not proof that product code has been completed.

## Product Scope

- Product name: 数据收益分配系统 / 数据要素收益分配软著产品。
- Product positioning: generic system for simulated data-element revenue allocation workflow demonstration and audit explanation.
- Medical scenarios: sample cases only; they must not define the generic system architecture.
- Data boundary: examples are simulated data and must not be described as real hospital data or real business settlement results.
