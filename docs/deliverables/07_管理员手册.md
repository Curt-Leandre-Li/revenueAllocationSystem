# 管理员手册

## 适用范围

本文为 V1.2 管理员手册归档摘要。当前 P0 为本地操作员模式，不要求登录、
生产级 RBAC、PDF、异步队列、多租户、真实付款、电子签章、税务或银行系统。

## 本地操作员

P0 默认操作人为 `local_operator`。本地操作员可完成演示数据/JSON 上传、
数据管理、质量评估、数元计量、贡献效用、MD-DShap、合同分配规则、收益分配模拟、
报告导出和审计查看。

## P1 管理能力

P1 可扩展登录、用户、角色、权限、PDF、CSV/XLSX 模板导入、异步任务进度、
历史报告管理和更完整权限控制；当前本地 P1 接口可用于演示闭环，不代表生产级安全能力。

## 管理注意事项

- 不上传真实敏感数据。
- 导出报告必须保留免责声明。
- 已确认/已导出项目不得直接覆盖修改，应复制新版本。
- 每次导出必须保留 `report_id` 和 `checksum`。
- 审计日志应能追溯输入、参数、输出、报告和算法快照。

系统结果仅为模拟参考，非法律结算 / 非法定结算结果。
