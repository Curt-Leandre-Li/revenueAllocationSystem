# 数据库设计

## 文档定位

本文为 V1.2 数据库设计归档摘要。详细设计以根目录
`数据收益分配系统_数据库设计与ER关系图_V1.0_导航结构更新版.md` 为准。

本文不是数据库迁移脚本，不代表已创建真实数据库表。

## 关键设计输入

- `nav_menu` 固化六个一级导航和二级页面。
- `permission` 支撑菜单、页面、按钮、导出权限。
- `module_code` 覆盖 SYS、DATA、RES、PARTY、QUAL、DU、UTIL、MDS、ALLOC、
  CONS、REP、PARAM、USER、AUD。
- `audit_log` 记录 `menu_code`、`module_code`、operation、operator、
  before/after、snapshot id、status、failure reason。
- `snapshot_store` 记录 INPUT、PARAMETER、RESULT、REPORT、ALGORITHM_AUDIT
  快照和 checksum。
- `report_record` 和 `export_file` 记录 `report_id`、文件、字段范围和
  checksum。

## P0/P1 边界

P0 可采用本地文件或轻量数据库映射同名对象；P1 可扩展正式关系数据库、
登录/RBAC、PDF、异步任务和历史报告管理。

系统结果仅为模拟参考，非法律结算 / 非法定结算结果。
